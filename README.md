# QR Code Generator

## Installation
```bash
git clone https://github.com/mustafadikyar/qr-generator-golang.git
cd qr-generator-golang
go mod tidy
```

## Configuration
Create your config:
```bash
width: 300           
height: 300          
content: https://mustafadikyar.com.tr  
outputDir: output
```

## Usage
```bash
go run cmd/main.go
```
